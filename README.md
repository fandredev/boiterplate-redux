## Clone this repo

###### Notes: Please, check the .tool-versions before use this project to show versions that i used.

```
git clone git@github.com:fandredev/boiterplate-redux.git
```

## Go to directory

```
 cd boiterplate-redux
```

## Install dependencies

```
 yarn
```

## Run server

```
 yarn run dev
```

## Run linter

```
 yarn run lint
```

## Run Build (OPTIONAL)

```
 yarn run build
```
